Autonomous UUV Environmental Monitoring | Grant-Funded Research

Challenge: Build an autonomous system to identify and track river pollution sources that impact ecosystems.

Solution: Developed a BlueROV2-based monitoring stack that won grant funding. Implemented a quadratic program with sub-meter trajectory tracking. Built an advection–diffusion modeling simulation in FiPy and an inverse Physics-Informed Neural Network (PINN) to infer diffusivity, flow field, and source parameters from a sparse set of samples. Implemented a novel residual map hotspot tracking map from the PINN to drive an online learning policy and generate optimal UUV trajectories.

Technologies: Convex Optimization • BlueROV2 • ArduSub • Fluid Dynamics • FiPy PDE Solver • Gazebo Simulation • CVXPY

Quadruped Robot Navigation | State Estimation in GPS-Denied Environments

Challenge: Enable autonomous navigation for the Unitree Go2 quadruped robot without access to GPS or visual odometry.

Solution: Developed a state estimator using an EKF to fuse IMU and magnetometer data for roll, pitch, and yaw estimation. Implemented a Model Predictive Controller based on unicycle dynamics, achieving trajectory tracking on complex paths including figure-eight patterns.

Impact: Demonstrated that navigation is achievable even with limited sensor suites, providing a cost-effective solution for indoor robotics applications. The system successfully executed complex trajectories.

Technologies: Kalman Filtering • Sensor Fusion • Model Predictive Control • ROS2 • Python • C++

Robotic Manipulation | Custom Computer Vision Pipeline

Challenge: Develop a system capable of identifying, classifying, and grasping objects of varying shapes using only a Realsense depth camera.

Solution: Engineered an end-to-end computer vision and manipulation pipeline processing point clouds. Implemented clustering algorithms for object segmentation, custom corner detection for precise orientation estimation, using my own algorithm which achieved a 85% success rate. Impact: Delivered a production-ready system, the architecture enables easy adaptation to new object types.

Technologies: Point Cloud Library (PCL) • MoveIt • Computer Vision • 3D Perception • ROS • Motion Planning
