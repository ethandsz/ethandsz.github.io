Quadruped Robot Navigation | Advanced State Estimation in GPS-Denied Environments

Challenge: Enable precise autonomous navigation for the Unitree Go2 quadruped robot without access to GPS, magnetometer, or visual odometry - simulating real-world scenarios in underground or indoor environments.

Solution: Architected a robust state estimation pipeline using Kalman filtering to fuse IMU sensor data for accurate roll, pitch, and yaw estimation. Overcame the critical challenge of yaw drift in GPS-denied environments by developing an innovative constant-yaw assumption algorithm validated through extensive odometry analysis. Implemented a Model Predictive Controller based on unicycle dynamics, achieving trajectory tracking with less than 10cm deviation on complex paths including figure-eight patterns.

Impact: Demonstrated that sophisticated navigation is achievable even with limited sensor suites, providing a cost-effective solution for indoor robotics applications. The system successfully executed complex trajectories that pushed the boundaries of what's possible without traditional localization methods.

Technologies: Kalman Filtering • Sensor Fusion • Model Predictive Control • ROS2 • Python • C++ • Unitree SDK • Real-time Control Systems
